Act. # 2

Formularios con PHP

> Descripcion

Se deberá crear un formulario en PHP, que almacene N datos (entre 1 a 50) en un arreglo. Se debe: - El formulario debe leer N. - Usar el paradigma Orientado a Objetos - Ordenar el Arreglo (Se debe seleccionar si se desea ordenar) - Obtener la Media y Varianza

> 2.- Aplicación

La aplicación comienza con un formulario, el cual tiene un 'input' de tipo texto, que nos permite ingresar el valor de 'N', además cuenta con una casilla o 'checkbox' que permite visualizar el arreglo ordenado de menor a mayor en caso de estar activada.

Para este ejemplo, se declaró 'N' de 10 elementos, sin la opción de ordenar el arreglo. Una vez declarado el arreglo, se procede a enviar el formulario y este redirige a una página que calcula y muestra los resultados, los cuales son mostrados de la siguiente forma:

En primera instancia, se enfatiza el valor de 'N', lo cuál permite tener recordar el tamaño del arreglo en todo momento. Como segundo apartado, se muestra el arreglo tal cuál como fué generado.

En las secciones inferiores se destacan los resultados de la Media (Promedio) y la Varianza. Al final, después de todo el cálculo, se muestra la cantidad de elementos que existen en el arreglo, cuyo máximo es el elemento mayor del arreglo base, el cuál cambia cada vez que es computado el proceso de llenado.

Como podemos observar, en este ejemplo, existen dos instancias del valor '50' (siendo éste el valor máximo), que se ven contadas de forma correcta al mostrar la cantidad de cada elemento del arreglo de la forma '50-->[2]'

> 3.- Proceso

El proceso comenzó haciendoce en archivos separados. siendo el formulario principal colocado dentro del archivo 'index.php' y separando así, cada esencial por separado, los cuales detallaremos a continuación.

En el archivo 'index.php' se creó un formulario para enviar los datos, en este caso solo la variable de tamaño 'N' y la verificación para ordenar el arreglo al archivo 'activity2.php'

Primero, cabe destacar que la sección de la sentencia 'if( preg_match('/[^0-9]/i'))' nos permite saber cuando se ingresa un caractér no numérico.

Dentro del archivo 'activity2.php' podemos destacar la sección anterior, que muestra la instanciación de la clase utilizada para el ejercicio sobresaltado por la línea roja. Después de la instanciación, se llama al método 'fillArray()' para que limpie y llene el arreglo con datos variables.

Posteriormente, la estructura de desición 'IF' nos permite saber cuando el usuario va a querer el arreglo ordenado. Justo después, se llama al método 'medVar()' para mostrar la Varianza y la Media

Por último, se destacan los 'else' de cada el manejo de errores en caso de que el usuario se equivoque

La clase más importante 'Datos' es la que contiene todos los métodos necesarios para que el programa funcione.

Usando como base la clase provista en la descripción, se modificó un poco para contener todos los métodos y atributos necesarios para que el programa funcionase de una manera óptima.

Los métodos primarios o más importantes son:

ordena() - Ordena el arreglo utilizando el algoritmo 'Count Sort'

cuenta() - Cuenta cada elemento y lo almacena en un array específico.

medVar() - Calcula y muestra la Media y Varianza de los datos provistos

muestra() - Muestra el Arreglo que recibe por parámetro
